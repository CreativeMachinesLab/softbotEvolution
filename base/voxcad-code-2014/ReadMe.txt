To use the visual studio solution, you must have the qt framework installed (with the visual studio plugin)

qt framework:
http://qt.nokia.com/products/

qt visual studio plugin:
http://qt.nokia.com/downloads/visual-studio-add-in


qwt (a qt plotting plugin) is also needed.
http://qwt.sourceforge.net/index.html
(svn repository: https://qwt.svn.sourceforge.net/svnroot/qwt/trunk/qwt)
