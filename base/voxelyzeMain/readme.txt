Basic 1
====================

Created: June 10, 2014



Description

This is a first example with using the voxelyze library.
It takes NickC's code (created back in 2013) and a simple
vxa file and compiles and runs a simulation.


Library dependencies:
o voxelyze.0.9.92



Useage:

Command line arguments are:
o 'f': specify the input filename
o 'p': print status during the simulation

$ voxelize -f Example_1.vxa -p
